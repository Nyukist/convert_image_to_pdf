# image_to_pdf

이미지 파일들을 pdf로 저장하는 스크립트입니다.

---

## 설치

이 저장소를 직접 받으셔야합니다. (로컬에 clone 하거나 zip 다운로드)

그 후, requirement.txt 의 라이브러리들을 설치하고 사용하시면 됩니다.

```bash
$ pip install --upgrade pip
$ pip install -r requirement.txt
```
